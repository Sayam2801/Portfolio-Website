Covid Tracker

In this project, I have extracted all the latest details related to Covid-19 from a third party API. The various technologies which I used in building this project are HTML5, CSS3, JavaScript, NodeJS, ExpressJS and EJS.

More

Sign Up & Login

This project is a traditional sign up and login web application which facilitates the user to not only signup and login but also search for a particular user. I developed this project using technologies like HTML5, CSS3, JavaScript, NodeJS, ExpressJS, Mongoose DB, MongoDB Atlas, JSON Web Token, etc.

More

Movie Seat Booking

This is a typical movie seat booking website developed primarily using HTML5, CSS3 and JavaScript ES6. User can select any movie among a given set of movies, select total number of seats in the theatre and the website will display the total price for watching the movie.
